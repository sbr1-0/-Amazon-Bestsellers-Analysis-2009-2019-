# -Amazon-Bestsellers-Analysis-2009-2019-

## 📌 Project Overview
This project explores a dataset of Amazon bestselling books from 2009 to 2019, with the goal of identifying patterns in popularity, pricing, and user satisfaction.

The analysis focuses on answering key business questions such as:
- What factors influence best-selling books?
- Are highly rated books also the most popular?
- How do Fiction and Non Fiction categories differ?

This project was developed as part of a data science learning portfolio, emphasizing data cleaning, exploratory data analysis (EDA), and business-oriented visualization.
